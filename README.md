# Bicep Curl Counter using YOLOv8 Pose

This project tracks and counts bicep curls using **YOLOv8 Pose Estimation**.

## 📋 Features
- Detect human pose using YOLOv8.
- Calculate elbow angles.
- Count reps based on arm movement.

## 🛠️ Requirements
- Python 3.8+
- Ultralytics YOLO
- OpenCV
- cvzone
- numpy

## 🚀 Run the project
```bash
python main.py
